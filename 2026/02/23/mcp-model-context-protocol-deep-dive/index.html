MCP (Model Context Protocol) жҠҖжңҜе®һзҺ°ж·ұеәҰи§Јжһҗ

еҸ‘иЎЁдәҺ 2026-02-23 | еҲҶзұ»дәҺ Technology | иҜ„и®әж•°пјҡ 0 | йҳ…иҜ»ж¬Ўж•°пјҡ

еүҚиЁҖ

2024е№ҙеә•пјҢAnthropicејҖжәҗдәҶ Model Context Protocol (MCP) вҖ”вҖ” дёҖдёӘиҝһжҺҘAIеә”з”ЁдёҺеӨ–йғЁзі»з»ҹзҡ„ејҖж”ҫж ҮеҮҶгҖӮMCPе°ұеғҸAIеә”з”Ёзҡ„вҖқUSB-CжҺҘеҸЈвҖқпјҢжҸҗдҫӣдәҶж ҮеҮҶеҢ–зҡ„ж–№ејҸи®©AIжЁЎеһӢи®ҝй—®ж•°жҚ®жәҗгҖҒи°ғз”Ёе·Ҙе…·гҖҒжү§иЎҢе·ҘдҪңжөҒгҖӮ

жң¬ж–Үе°Ҷж·ұе…Ҙеү–жһҗMCPзҡ„жҠҖжңҜжһ¶жһ„е’Ңж ёеҝғе®һзҺ°еҺҹзҗҶгҖӮ

д»Җд№ҲжҳҜMCPпјҹ

MCPжҳҜдёҖдёӘејҖж”ҫеҚҸи®®пјҢе®ғе®ҡд№үдәҶAIеә”з”ЁдёҺеӨ–йғЁзі»з»ҹд№Ӣй—ҙзҡ„йҖҡдҝЎж ҮеҮҶгҖӮйҖҡиҝҮMCPпјҡ

  • AIеә”з”ЁпјҲеҰӮClaudeгҖҒChatGPTпјүеҸҜд»ҘиҝһжҺҘж•°жҚ®жәҗпјҲж–Үд»¶гҖҒж•°жҚ®еә“пјү
  • и°ғз”Ёе·Ҙе…·пјҲжҗңзҙўеј•ж“ҺгҖҒи®Ўз®—еҷЁпјү
  • жү§иЎҢе·ҘдҪңжөҒпјҲзү№е®ҡжҸҗзӨәжЁЎжқҝпјү

ж ёеҝғжһ¶жһ„

дёүз§ҚеҸӮдёҺиҖ…

MCPйҮҮз”Ёе®ўжҲ·з«Ҝ-жңҚеҠЎеҷЁжһ¶жһ„пјҢеҢ…еҗ«дёүз§Қи§’иүІпјҡ

и§’иүІ иҜҙжҳҺ зӨәдҫӢ
Host AIеә”з”ЁпјҢеҚҸи°ғеӨҡдёӘMCPе®ўжҲ·з«Ҝ Claude Desktop, VS Code
Client з»ҙжҠӨдёҺеҚ•дёӘMCP Serverзҡ„иҝһжҺҘ HostеҶ…зҡ„иҝһжҺҘз»„д»¶
Server жҸҗдҫӣдёҠдёӢж–ҮгҖҒе·Ҙе…·е’Ңиө„жәҗ ж–Үд»¶зі»з»ҹжңҚеҠЎгҖҒGitHubжңҚеҠЎ

дёӨеұӮз»“жһ„

MCPеҚҸи®®еҲҶдёәдёӨеұӮпјҡ

1. ж•°жҚ®еұӮ (Data Layer)

еҹәдәҺ JSON-RPC 2.0 зҡ„еҚҸи®®еұӮпјҢе®ҡд№үпјҡ

  • з”ҹе‘Ҫе‘Ёжңҹз®ЎзҗҶпјҡиҝһжҺҘеҲқе§ӢеҢ–гҖҒиғҪеҠӣеҚҸе•ҶгҖҒиҝһжҺҘз»Ҳжӯў
  • жңҚеҠЎеҷЁзү№жҖ§пјҡToolsгҖҒResourcesгҖҒPrompts
  • е®ўжҲ·з«Ҝзү№жҖ§пјҡSamplingгҖҒElicitationгҖҒLogging
  • е·Ҙе…·зү№жҖ§пјҡйҖҡзҹҘгҖҒиҝӣеәҰиҝҪиёӘ

2. дј иҫ“еұӮ (Transport Layer)

з®ЎзҗҶйҖҡдҝЎйҖҡйҒ“е’Ңи®ӨиҜҒпјҡ

  • Stdio Transportпјҡжң¬ең°иҝӣзЁӢйҖҡдҝЎ
  • Streamable HTTP TransportпјҡиҝңзЁӢжңҚеҠЎеҷЁйҖҡдҝЎ

дј иҫ“еұӮиҜҰи§Ј

Stdio TransportпјҲж ҮеҮҶиҫ“е…Ҙиҫ“еҮәпјү

йҖӮз”ЁдәҺжң¬ең°MCP Serverпјҡ

зү№зӮ№пјҡ

  • ClientеҗҜеҠЁServerдҪңдёәеӯҗиҝӣзЁӢ
  • йҖҡиҝҮstdin/stdoutдәӨжҚўJSON-RPCж¶ҲжҒҜ
  • ж¶ҲжҒҜд»ҘжҚўиЎҢз¬ҰеҲҶйҡ”
  • ж— зҪ‘з»ңејҖй”ҖпјҢжҖ§иғҪжңҖдјҳ
  • stderrеҸҜз”ЁдәҺж—Ҙеҝ—иҫ“еҮә

Streamable HTTP Transport

йҖӮз”ЁдәҺиҝңзЁӢMCP Serverпјҡ

зү№зӮ№пјҡ

  • дҪҝз”ЁHTTP POSTеҸ‘йҖҒж¶ҲжҒҜ
  • ж”ҜжҢҒServer-Sent Events (SSE)иҝӣиЎҢжөҒејҸе“Қеә”
  • ж”ҜжҢҒж ҮеҮҶHTTPи®ӨиҜҒпјҲBearer TokenгҖҒAPI Keyпјү
  • жҺЁиҚҗдҪҝз”ЁOAuthиҺ·еҸ–и®ӨиҜҒд»ӨзүҢ
  • ж”ҜжҢҒдјҡиҜқз®ЎзҗҶпјҲMcp-Session-Idпјү

дјҡиҜқз®ЎзҗҶжөҒзЁӢпјҡ

ж ёеҝғеҺҹиҜӯ (Primitives)

MCPе®ҡд№үдәҶдёүз§ҚжңҚеҠЎеҷЁз«ҜеҺҹиҜӯпјҡ

1. ToolsпјҲе·Ҙе…·пјү

еҸҜжү§иЎҢеҮҪж•°пјҢAIжЁЎеһӢеҸҜи°ғз”Ёжү§иЎҢж“ҚдҪңпјҡ

2. ResourcesпјҲиө„жәҗпјү

жҸҗдҫӣдёҠдёӢж–Үж•°жҚ®зҡ„ж•°жҚ®жәҗпјҡ

  • ж–Үд»¶еҶ…е®№
  • ж•°жҚ®еә“и®°еҪ•
  • APIе“Қеә”

3. PromptsпјҲжҸҗзӨәжЁЎжқҝпјү

еҸҜеӨҚз”Ёзҡ„дәӨдә’жЁЎжқҝпјҡ

е®ўжҲ·з«ҜеҺҹиҜӯ

MCPиҝҳе®ҡд№үдәҶе®ўжҲ·з«ҜеҸҜжҸҗдҫӣзҡ„еҺҹиҜӯпјҢи®©жңҚеҠЎеҷЁиғҪжһ„е»әжӣҙдё°еҜҢзҡ„дәӨдә’пјҡ

SamplingпјҲйҮҮж ·пјү

е…Ғи®ёжңҚеҠЎеҷЁиҜ·жұӮLLMиЎҘе…ЁпјҢе®һзҺ°жңҚеҠЎеҷЁз«Ҝзҡ„AIиғҪеҠӣпјҡ

з”ЁйҖ”пјҡжңҚеҠЎеҷЁйңҖиҰҒAIиғҪеҠӣдҪҶдёҚжғіеҶ…зҪ®жЁЎеһӢSDK

ElicitationпјҲиҜўй—®пјү

е…Ғи®ёжңҚеҠЎеҷЁиҜ·жұӮз”ЁжҲ·жҸҗдҫӣйўқеӨ–дҝЎжҒҜпјҡ

з”ҹе‘Ҫе‘Ёжңҹз®ЎзҗҶ

MCPжҳҜжңүзҠ¶жҖҒеҚҸи®®пјҢйңҖиҰҒе®Ңж•ҙзҡ„з”ҹе‘Ҫе‘Ёжңҹз®ЎзҗҶпјҡ

еҲқе§ӢеҢ–жҸЎжүӢ

иғҪеҠӣеҚҸе•Ҷ

capabilitiesеҜ№иұЎеЈ°жҳҺж”ҜжҢҒзҡ„еҠҹиғҪпјҡ

е®ўжҲ·з«ҜиғҪеҠӣпјҡ

  • elicitation: ж”ҜжҢҒз”ЁжҲ·дәӨдә’иҜ·жұӮ
  • sampling: ж”ҜжҢҒLLMйҮҮж ·

жңҚеҠЎеҷЁиғҪеҠӣпјҡ

  • tools: ж”ҜжҢҒе·Ҙе…·и°ғз”Ё
  • resources: ж”ҜжҢҒиө„жәҗи®ҝй—®
  • prompts: ж”ҜжҢҒжҸҗзӨәжЁЎжқҝ
  • listChanged: ж”ҜжҢҒеҲ—иЎЁеҸҳжӣҙйҖҡзҹҘ

е·Ҙе…·еҸ‘зҺ°дёҺжү§иЎҢ

еҸ‘зҺ°е·Ҙе…·

жү§иЎҢе·Ҙе…·

йҖҡзҹҘжңәеҲ¶

MCPж”ҜжҢҒе®һж—¶йҖҡзҹҘпјҢе®һзҺ°еҠЁжҖҒжӣҙж–°пјҡ

еҪ“жңҚеҠЎеҷЁзҡ„е·Ҙе…·еҲ—иЎЁеҸ‘з”ҹеҸҳеҢ–ж—¶пјҢжңҚеҠЎеҷЁеҸҜд»Ҙдё»еҠЁйҖҡзҹҘе®ўжҲ·з«ҜеҲ·ж–°гҖӮ

е®үе…ЁиҖғйҮҸ

MCPиөӢдәҲAIејәеӨ§зҡ„иғҪеҠӣпјҢе®үе…ЁиҮіе…ійҮҚиҰҒпјҡ

ж ёеҝғеҺҹеҲҷ

  1. з”ЁжҲ·еҗҢж„ҸдёҺжҺ§еҲ¶

    • з”ЁжҲ·еҝ…йЎ»жҳҺзЎ®еҗҢж„ҸжүҖжңүж•°жҚ®и®ҝй—®е’Ңж“ҚдҪң
    • з”ЁжҲ·дҝқз•ҷеҜ№е…ұдә«ж•°жҚ®е’Ңжү§иЎҢж“ҚдҪңзҡ„жңҖз»ҲжҺ§еҲ¶жқғ

  2. ж•°жҚ®йҡҗз§Ғ

    • Hostеҝ…йЎ»иҺ·еҫ—жҳҺзЎ®еҗҢж„ҸжүҚиғҪеҗ‘жңҚеҠЎеҷЁжҡҙйңІз”ЁжҲ·ж•°жҚ®
    • жңӘз»ҸеҗҢж„ҸдёҚеҫ—дј иҫ“иө„жәҗж•°жҚ®

  3. е·Ҙе…·е®үе…Ё

    • е·Ҙе…·д»ЈиЎЁд»»ж„Ҹд»Јз Ғжү§иЎҢпјҢеҝ…йЎ»и°Ёж…ҺеҜ№еҫ…
    • и°ғз”Ёд»»дҪ•е·Ҙе…·еүҚеҝ…йЎ»иҺ·еҫ—жҳҺзЎ®еҗҢж„Ҹ

  4. LLMйҮҮж ·жҺ§еҲ¶

    • з”ЁжҲ·еҝ…йЎ»жҳҺзЎ®жү№еҮҶд»»дҪ•LLMйҮҮж ·иҜ·жұӮ
    • з”ЁжҲ·жҺ§еҲ¶йҮҮж ·зҡ„жҸҗзӨәе’ҢеҸҜи§Ғз»“жһң

жҖ»з»“

MCPйҖҡиҝҮзІҫе·§зҡ„еҚҸи®®и®ҫи®ЎпјҢе®һзҺ°дәҶпјҡ

  1. ж ҮеҮҶеҚҸи®®пјҡеҹәдәҺJSON-RPC 2.0зҡ„з»ҹдёҖйҖҡдҝЎеҚҸи®®
  2. еҸҢеұӮжһ¶жһ„пјҡж•°жҚ®еұӮдёҺдј иҫ“еұӮжё…жҷ°еҲҶзҰ»
  3. дё°еҜҢеҺҹиҜӯпјҡToolsгҖҒResourcesгҖҒPromptsжҸҗдҫӣе®Ңж•ҙзҡ„дёҠдёӢж–ҮиғҪеҠӣ
  4. еҸҢеҗ‘йҖҡдҝЎпјҡе®ўжҲ·з«ҜеҺҹиҜӯи®©жңҚеҠЎеҷЁд№ҹиғҪдё»еҠЁдәӨдә’
  5. е®үе…Ёдјҳе…Ҳпјҡд»ҺеҚҸи®®и®ҫи®ЎеұӮйқўејәи°ғз”ЁжҲ·жҺ§еҲ¶
